 # RubberDocker - Docker From Scratch

This repo is a follow-along of the original [RubberDocker](https://github.com/tonistiigi/rubberdocker) by Tonis Tiigi.

In this project, I recreate the workshop exercises to better understand Linux process creation, the fork & exec model, software threads, and the basics of container-style isolation. I’m exploring Docker from the ground up, constructing it step by step to understand how it works.
